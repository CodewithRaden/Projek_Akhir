Hear What We Have To Say ...

This project is a smart locker system that allows users to access lockers using RFID or PIN-based authentication. It logs access times and maintains a real-time status of locker availability, integrating hardware and software components for secure and user-friendly access control.

This project was developed as part of the Indobot Academy Independent Study Program, which provides students with hands-on experience in IoT and technology innovation

We created the smart locker with RFID and keypad access to provide a secure, efficient, and flexible storage solution. By combining RFID and PIN authentication, it ensures only authorized access while offering user convenience. The system includes real-time monitoring via a web interface, making it ideal for shared environments like schools, gyms, or offices.

This project has been developed gradually over the past few months as the timeline of MSIB Independent Study under Indobot Academy. We built and refined different components, working on the hardware, web interface, and database integration in phases.

The smart locker system can be implemented in locations that require secure, user-specific access to shared storage, such as schools, gyms, coworking spaces, and libraries. For this project, we plan to implement it on a campus.

The project combines hardware (ESP32 with RFID module) and software (Python Flask for the web app). It includes a registration system for users, RFID authentication for access, and real-time status monitoring. The system stores data in a Firebase database and integrates additional features, such as admin controls, for user and locker management.
